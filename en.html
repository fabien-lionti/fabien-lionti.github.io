Some personal projects:

API for Handwritten Text Recognition

This API recognizes handwritten or typed content and converts it into text. It is developed in Python using the OpenCV, TensorFlow, and Flask libraries. A web application was also implemented using HTML5/CSS3 and JavaScript to interface with the API.

Particle Swarm Optimization

Particle Swarm Optimization (PSO) is an evolutionary algorithm inspired by the collective behavior of bird flocks and fish schools. It is a metaheuristic that does not require the problem to be continuous. I implemented this algorithm to optimize hyperparameter combinations for a kernel used in training a Support Vector Machine (SVM).

Least Squares Support Vector Machine
